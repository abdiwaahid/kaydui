## ✨ Features

- 🎨 **100+ Beautiful Components** - Production-ready UI components built with Tailwind CSS
- 📋 **Copy & Paste** - No installation required, just copy and use
- 🌙 **Dark Mode Support** - All components work seamlessly in light and dark themes
- 📱 **Fully Responsive** - Mobile-first design that looks great on all devices
- ⚡ **Zero Dependencies** - Pure HTML, CSS, and vanilla JavaScript
- 🔍 **Advanced Search** - Find components quickly with smart filtering
- ❤️ **Favorites System** - Save your most-used components
- 🎯 **Production Ready** - Accessible, optimized, and battle-tested
- 🔧 **Highly Customizable** - Easy to modify colors, spacing, and styling
- 📖 **Comprehensive Docs** - Detailed usage instructions for every component

## 🛠️ Usage

### Basic Usage
1. Browse components on [kaydui](https://abdiwaahid.github.io/kaydui/)
2. Click the "Copy" button on any component
3. Paste the HTML into your project
4. Ensure Tailwind CSS is included in your project

## 🤝 Contributing

We welcome contributions! Here's how you can help:

### Adding New Components
1. Fork the repository
2. Add your component to corresponding categroy in components folder 
4. Add appropriate tags and category
5. Test in both light and dark modes
6. Submit a pull request

### Component Structure
\`\`\`javascript
{
  id: "unique-id",
  slug: "component-name",
  title: "Component Title",
  description: "Brief description of the component",
  category: "Category Number",
  tags: ["tag1", "tag2", "tag3"],
  code: `<!-- Your component HTML -->`,
}
\`\`\`

### Guidelines
- ✅ Use semantic HTML
- ✅ Follow accessibility best practices
- ✅ Test in multiple browsers
- ✅ Ensure mobile responsiveness
- ✅ Include hover and focus states
- ✅ Use consistent naming conventions
